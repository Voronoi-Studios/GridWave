# TileSet Collection Nodes

Bundles multiple TileSet Nodes together for easier export and import.
Can be read from in the folder `Server/HytaleGenerator/TileSetCollecions` 

## Variants

---

### Automatic TileSet Collection
<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/AutomaticTileSetCollection.png" alt="AutomaticTileSetCollection" />

Automatically creates all TileSets based on the folders naming, allows for export 

---

### Imported TileSet Collection

<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/ImportedTileSetCollection.png" alt="ImportedTileSetCollection" />

Allows you to import nodes

---

### TileSet Collection
<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/SimpleTileSetCollection.png" alt="SimpleTileSetCollection"/> 

Standard node, allows for exports

---

### Union TileSet Collection
<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/UnionTileSetCollection.png" alt="UnionTileSetCollection" />

Combines the TileSets in to one fresh TileSetCollection
