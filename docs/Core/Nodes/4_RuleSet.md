# RuleSet Nodes

Defines each edge of a cell
Can be read from in the folder `Server/HytaleGenerator/RuleSets` 

## Variants

---

### Simple 2D RuleSet

<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/Simple2DRuleSet.png" alt="Simple2DRuleSet" />

Use some sort of key to define what the corresponding edge can connect to.
Multiple keys can be entered by using `,` as a separator

### Advanced 2D RuleSet
<img class="node" src="https://github.com/Voronoi-Studios/GridWave/raw/main/docs/Images/Nodes/Advanced2DRuleSet.png" alt="Advanced2DRuleSet" />

Use some sort of key to define what the corresponding edge can connect to.
The advanced variant allows you to specify separately "Who am I" => Provider RuleSets and "Who do I like" => Reciever RuleSets.

Multiple keys can be entered by using `,` as a separator
